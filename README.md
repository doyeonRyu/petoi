## 폴더별 설명
1. etc   
- Langchain 공부한 자료   
- LangChain 정리.docx: lang chain 공부한 자료   
- langchain.ipynb: lang chain 코드 실습   
- SQLite.ipynb: SQLite 코드 실습   
- text, user 파일: 실습 예제 결과들   

2. GPT_related   
- bittle_profile.json: 사용자 프로필 저장소   
- Bittle.db: 대화 이력 저장소   
- Commands.json: 대화, 명령어 매칭을 위한 명령어 정리   

3. Image   
- petoi_python_code/n14_camera.py 실행 결과   
- 객체 인식 후 이미지와 박스 저장   

4. petoi_arduino_code   
- send_image_to_python.ino   
- petoi_python_code/n14_camera.py와 함께 실행   
- 카메라 센서 작동 및 파이썬으로 정보 전달   
- 해당 폴더가 아닌 아래 서술된 폴더에서 실행해야 함   

5. petoi_python_code   
- 각 기술, 센서 작동 샘플 코드   
- 00: 파일 템플릿   
- 01 - 12: 기본, 추가 센서 작동 기본 예제   
- 13: GPT와 LangChain, SQLite를 통한 대화 시스템 구현   
    - 13_v2: DB 시스템이 아닌 json 버전 (v1 권장)
    - Speech2Text: STT 모듈   
    - Text2Speech: TTS 모듈
